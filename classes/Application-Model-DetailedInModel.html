Detailed In Model

This object model is for the "Detailed In" module which is equivalent to the assigned software in each Job or Project. Example of "Detailed In" software such as Tekla 2017, Tekla 2018, Tekla 2019, Tekla 2020, SDS2 2017, SDS2 2018, SDS2 2019, SDS2 2020, etc. It's intended for all database activities of Detailed In Model such as SELECT, UPDATE, DELETE and INSERT.
