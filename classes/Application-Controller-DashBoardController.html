Dashboard Controller

This object contains the summary and report monitoring in Dashboard. Such as the following;

  • Root Cause Monitoring
  • Manpower with Issue
  • Tardiness Monitoring
  • Leave Balance
  • Reimbursement
  • Reports
  • Non-Compliance Logs
  • Scheduled Suspension
  • Engineering Delinquent Timesheet Monitoring
  • Scheduled LeavesMemorandum & News/Articles
  • Popup notifications (Personal Memo, Health Declaration, Health on and Insurance)
